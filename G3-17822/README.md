# 17822 원판 돌리기

- https://www.acmicpc.net/problem/17822
- Implementation
---
1. 회전
    - 기존 원판 배열 복사 한 배열에서 위치 바꿔서 기존 원판에 넣음
    - 반시계 k 회전은 시계 M - k 회전
2. 업데이트
    - 같은 것 지우기
        - 인접 위치에 같은 것 있으면 그 위치부터 bfs
        - bfs 한 횟수 체크해서 0 이면 다음
    - 평균보다 크면 - 1, 작으면 + 1
        - 평균과 같으면 그대로
    - 같은 원판에서 처음과 끝이 인접함에 주의
